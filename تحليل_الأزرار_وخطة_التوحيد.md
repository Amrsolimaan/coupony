## 📐 التصميم المقترح

### الهيكل الهرمي:

```
AppButton (abstract base)
├── AppPrimaryButton (filled)
├── AppOutlinedButton (outlined)
└── AppTextButton (text only)
```

### الخصائص المشتركة (Base Class):

```dart
abstract class AppButton extends StatelessWidget {
  // Common properties
  final String text;
  final VoidCallback? onPressed;
  final bool isLoading;
  final bool isEnabled;
  final ButtonSize size;
  final double? width;
  final IconData? icon;
  final IconPosition? iconPosition;
  
  // Common methods
  double get height;
  double get fontSize;
  double get borderRadius;
  TextStyle get textStyle;
  Widget buildLoadingIndicator();
}
```
