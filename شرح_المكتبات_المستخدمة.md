## 🌐 الشبكة والاتصال (Networking)

### 6. dio: ^5.9.1
**الوصف:** مكتبة HTTP قوية ومتقدمة
**الأهمية:** أفضل من http العادية - تدعم Interceptors, Timeout, Retry
**المميزات:**
- إلغاء الطلبات
- رفع وتحميل الملفات مع progress
- معالجة الأخطاء بشكل احترافي
**الاستخدام:**
```dart
final response = await dio.get('https://api.example.com/data');
```

### 7. connectivity_plus: ^7.0.0
**الوصف:** فحص نوع الاتصال بالإنترنت
**الأهمية:** معرفة إذا كان المستخدم متصل بـ WiFi أو Mobile Data أو لا
**الاستخدام:**
```dart
final connectivityResult = await Connectivity().checkConnectivity();
if (connectivityResult == ConnectivityResult.mobile) {
  // متصل بالموبايل
}
```

### 8. internet_connection_checker: ^1.0.0+1
**الوصف:** التحقق الفعلي من وجود إنترنت
**الأهمية:** connectivity_plus تفحص الاتصال فقط، هذه تفحص الإنترنت الفعلي
**الفرق:** قد تكون متصل بـ WiFi لكن بدون إنترنت
**الاستخدام:**
```dart
bool hasInternet = await InternetConnectionChecker().hasConnection;
```

## 💾 التخزين المحلي (Local Storage)

### 9. hive: ^2.2.3
**الوصف:** قاعدة بيانات NoSQL سريعة جداً مكتوبة بـ Dart
**الأهمية:** أسرع من SQLite بـ 10 مرات، لا تحتاج native code
**المميزات:**
- سرعة فائقة
- تشفير البيانات
- تخزين أي نوع من الكائنات
**الاستخدام:**
```dart
var box = await Hive.openBox('myBox');
box.put('name', 'Ahmed');
String name = box.get('name');
```
**متى تستخدمها:** للتخزين السريع والكائنات المعقدة

### 10. hive_flutter: ^1.1.0
**الوصف:** دمج Hive مع Flutter
**الأهمية:** يوفر path provider تلقائي ودعم Flutter
**الاستخدام:**
```dart
await Hive.initFlutter();
```

### 11. flutter_secure_storage: ^10.0.0
**الوصف:** تخزين آمن ومشفر للبيانات الحساسة
**الأهمية:** يستخدم Keychain في iOS و KeyStore في Android
**متى تستخدمها:** لتخزين tokens, passwords, API keys
**الاستخدام:**
```dart
final storage = FlutterSecureStorage();
await storage.write(key: 'token', value: 'my_secret_token');
String token = await storage.read(key: 'token');
```
**ملاحظة:** البيانات لا تُحذف حتى لو حذفت التطبيق

### 12. shared_preferences: ^2.5.3
**الوصف:** تخزين بسيط للإعدادات (key-value)
**الأهمية:** سريع وبسيط للإعدادات الصغيرة
**متى تستخدمها:** للإعدادات البسيطة (theme, language, first_time)
**الاستخدام:**
```dart
SharedPreferences prefs = await SharedPreferences.getInstance();
await prefs.setBool('isDarkMode', true);
bool isDark = prefs.getBool('isDarkMode') ?? false;
```

## 🛠️ الأدوات المساعدة (Utilities)

### 13. dartz: ^0.10.1
**الوصف:** مكتبة البرمجة الوظيفية (Functional Programming)
**الأهمية:** تساعد في معالجة الأخطاء بشكل احترافي
**المفاهيم الأساسية:**
- **Either<Left, Right>:** إما نجاح أو فشل
- **Option:** قيمة موجودة أو null
**الاستخدام:**
```dart
Either<Failure, User> result = await getUserData();
result.fold(
  (failure) => print('Error: $failure'),
  (user) => print('Success: ${user.name}'),
);
```
**متى تستخدمها:** في Clean Architecture لمعالجة الأخطاء

### 14. equatable: ^2.0.8
**الوصف:** تسهيل مقارنة الكائنات
**الأهمية:** بدونها، كل object له reference مختلف حتى لو البيانات نفسها
**الاستخدام:**
```dart
class User extends Equatable {
  final String name;
  final int age;
  
  @override
  List<Object> get props => [name, age];
}

User user1 = User('Ahmed', 25);
User user2 = User('Ahmed', 25);
print(user1 == user2); // true بدلاً من false
```
**متى تستخدمها:** مع BLoC States و Models

### 15. get_it: ^9.2.0
**الوصف:** Service Locator - حقن التبعيات (Dependency Injection)
**الأهمية:** تسهل الوصول للـ services من أي مكان بدون context
**المميزات:**
- Singleton
- Lazy Singleton
- Factory
**الاستخدام:**
```dart
// التسجيل
final getIt = GetIt.instance;
getIt.registerSingleton<ApiService>(ApiService());

// الاستخدام
final api = getIt<ApiService>();
```
**متى تستخدمها:** في Clean Architecture لحقن Repositories و UseCases

### 16. logger: ^2.6.2
**الوصف:** طباعة logs منظمة وملونة
**الأهمية:** تسهل debugging وتتبع الأخطاء
**المميزات:**
- ألوان مختلفة حسب المستوى
- طباعة Stack Trace
- تنسيق JSON
**الاستخدام:**
```dart
var logger = Logger();
logger.d('Debug message');
logger.i('Info message');
logger.w('Warning message');
logger.e('Error message');
```

### 17. crypto: ^3.0.7
**الوصف:** مكتبة التشفير والهاش
**الأهمية:** تشفير البيانات وإنشاء hash
**الاستخدام:**
```dart
import 'package:crypto/crypto.dart';
var bytes = utf8.encode('password123');
var hash = sha256.convert(bytes);
print(hash); // hash مشفر
```
**متى تستخدمها:** لتشفير passwords أو إنشاء unique IDs

## 📊 ملخص المعمارية المستخدمة

بناءً على المكتبات، المشروع يستخدم:

### Clean Architecture
- **Data Layer:** Dio + Hive
- **Domain Layer:** Dartz (Either) + Entities
- **Presentation Layer:** BLoC + Provider

### Dependency Injection
- GetIt لحقن التبعيات

### State Management
- BLoC للحالات المعقدة
- Provider للحالات البسيطة

### Error Handling
- Dartz Either<Failure, Success>
- Try-Catch مع Logger
